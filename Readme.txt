1. La racine du projet doit être 'localhost\vscar\'
2. pour lancer le projet il faut extraire le code dans un dossier 'vscar' et le mettre xampp/htdocs
3. vous devez être connecté à Internet pour que les liens bootstrap fonctionnent
3. Toutes les fonctionnalités présentées dans les spécifications sans mise en œuvre
4. J'ai ajouté des fonctionnalités optionnelles telles que :
        - aimer une marque
        - gestion des publicités
        - envoyer des messages contacts à l'administrateur

5. NOTE TRES IMPORTANTE : il faut modifier les paramètres d'accés à la base de données dans (model\DataBase.php):
        5.1. le changement de port est trés important pour que la connexion à Xampp fonctionne
        5.2. il faut nommer la bdd dans PHPMYADMIN tels que dans les paramètres de connexion à la base de données
